# 🐾 Paw Patrol by Sofi

Un sitio web dedicado a promover y facilitar la adopción de perros. Esta plataforma ofrece una experiencia visual, estructurada y accesible para conectar a las mascotas con sus futuros hogares, destacando perfiles detallados, galerías fotográficas y un proceso de solicitud claro.

## 🚀 Demo en Vivo
Puedes visitar el proyecto desplegado aquí: **[Paw Patrol by Sofi](https://paw-patrol-by-sofi.vercel.app)**

## ✨ Características
* **Galería de Adopciones:** Visualización atractiva de los perros que están buscando un hogar.
* **Perfiles Detallados:** Páginas individuales para cada mascota (como el perfil de Lola) detallando su historia y necesidades.
* **Formulario de Solicitud:** Interfaz clara y estructurada para que los interesados puedan enviar sus datos de adopción.
* **Diseño Accesible e Inclusivo:** Estructura semántica pensada para cumplir con estándares de accesibilidad, asegurando que la web sea navegable por cualquier usuario.

## 🛠️ Tecnologías Utilizadas
El proyecto está construido íntegramente con tecnologías fundamentales de frontend, garantizando rapidez y ligereza:
* **HTML5:** Estructura y semántica.
* **CSS3:** Estilos, maquetación modular y diseño responsivo adaptado a diferentes pantallas.

## 📂 Estructura Principal
El proyecto divide sus vistas y estilos para un mejor mantenimiento:
* `index.html` - Página de inicio / Landing page.
* `index_galery.html` - Catálogo completo de mascotas.
* `perfil_lola.html` - Vista en detalle del perfil de un animal.
* `index_formulario.html` - Formulario de contacto y adopción.
* Archivos `.css` modulares (ej. `styles_galery.css`, `perfil-animal.css`, `style_formulario.css`) para cada sección.
